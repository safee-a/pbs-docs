## Why HealthDocs.ai?

The PBS API v3 is becoming the **sole data source** for PBS data by May 2026, replacing deprecated XML and text downloads. But the official documentation is fragmented across multiple domains with no entity-relationship documentation.

## Quick Start

1. **No auth required** — the public PBS API needs no API key
2. **Get the current schedule:** `GET /schedules?get_latest_schedule_only=true`
3. **Search items:** `GET /items?schedule_code={code}&drug_name={name}`
4. **Respect the rate limit:** 1 request per 20 seconds, shared across all users
